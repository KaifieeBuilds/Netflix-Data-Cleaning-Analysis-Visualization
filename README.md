# Netflix Data: Cleaning, Analysis and Visualization

## Project Overview
This project focuses on cleaning, analyzing, and visualizing Netflix content data
to understand content distribution, popular genres, ratings, country-wise trends,
and content growth over time.

Exploratory Data Analysis (EDA) techniques are used to extract meaningful insights
from real-world Netflix data.

## Objectives
- Clean and preprocess the Netflix dataset
- Analyze Movies vs TV Shows distribution
- Identify popular genres and ratings
- Study country-wise content distribution
- Analyze content addition trends over the years
- Visualize insights using charts and plots

## Dataset
- Netflix Titles Dataset
- Time period: 2008–2021
- Domain: Entertainment / Media Analytics

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- WordCloud
- VS Code

## Analysis Performed
- Data cleaning and preprocessing
- Content type analysis (Movies vs TV Shows)
- Genre analysis
- Ratings analysis
- Country-wise content analysis
- Year-wise content growth analysis
- Word cloud visualization of movie titles

## Files in This Repository
- `netflix_analysis.ipynb` – Complete data analysis notebook

## Conclusion
The analysis reveals that Netflix has a larger number of movies than TV shows,
with drama and international content being the most dominant genres. The United
States and India contribute the highest number of titles, and Netflix content
additions have grown significantly over recent years.
